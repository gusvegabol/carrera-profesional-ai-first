# 04_patrones_de_preguntas

Esta carpeta reunirá patrones, secuencias y familias de preguntas para la entrevista profesional.

- Organizar preguntas por intención.
- Saber cuándo profundizar.
- Evitar bancos de preguntas planos.

## Contenido futuro

- Preguntas de contexto.
- Preguntas sobre tareas.
- Preguntas sobre problemas.
- Preguntas sobre decisiones.
- Preguntas sobre logros.
- Preguntas de validación.
