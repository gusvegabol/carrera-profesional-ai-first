# 00_proposito

Esta carpeta contiene la base fundacional de la bóveda de entrevista profesional.

## Función

- Explicar por qué existe la bóveda.
- Definir su enfoque inicial.
- Alinear principios antes de entrar en investigación o síntesis.

## Contenido futuro

- Idea central.
- Preguntas guía.
- Principios iniciales.

## Relación con el playbook

Todo lo que se escriba más adelante en la bóveda debería poder rastrearse hasta esta base. Si una idea no nace de aquí o no encaja con el propósito, probablemente no pertenece a la bóveda.
