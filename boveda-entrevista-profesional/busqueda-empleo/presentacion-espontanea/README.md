# Materiales para candidatura espontánea

Esta carpeta reúne materiales generales de Job-up para una presentación espontánea. No corresponde a una candidatura para una oferta concreta.

Los tres artefactos base son:

- `cv-maestro.docx`
- `cv-maestro.pdf`
- [[email-presentacion]]

Solo si un destinatario lo solicita, podrán crearse `carta-presentacion.docx` y `carta-presentacion.pdf`.

Todo envío o compartición de estos materiales requiere la aprobación humana explícita de la persona candidata.
