# 08_entrevistas_piloto

Esta carpeta reúne entrevistas piloto reales o semi-reales usadas para validar el método de entrevista profesional profunda.

## Convención de nombres

Los documentos de esta carpeta deben usar la raíz `PILOTO_`.

Convención recomendada: `PILOTO_###_ENTREVISTA_<CONTEXTO>.md`.

Ejemplo: `PILOTO_001_ENTREVISTA_HERFRAILES_RRHH.md`.

## Diferencia con `06_ejemplos/`

- `06_ejemplos/` contiene material didáctico, sintético, anotado o calibrador.
- `08_entrevistas_piloto/` contiene casos piloto con experiencia concreta y trazabilidad de validación.

## Criterios de inclusión

- La entrevista es real o semi-real.
- Se usa para probar el método, no solo para ilustrarlo.
- Hay evidencia suficiente para justificar su registro.
- El documento no es una plantilla genérica ni un ejemplo didáctico puro.

## Regla de cautela

Un piloto valida parcialmente el método, no lo demuestra de forma definitiva.
