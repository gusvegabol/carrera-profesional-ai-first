# 08_fuentes

Esta carpeta guardará referencias, notas y material de apoyo para la investigación futura.

## Función

- Centralizar apoyos.
- Mantener trazabilidad de lecturas y referencias.
- Separar fuente de síntesis.

## Contenido futuro

- Libros.
- Artículos.
- Papers.
- Vídeos.
- Notas de investigación.

## Relación con el playbook

Las fuentes alimentan la investigación, pero el playbook debe salir de una síntesis propia y no de una simple colección de enlaces.
