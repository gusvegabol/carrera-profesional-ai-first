# 06_ejemplos

Esta carpeta servirá para mostrar ejemplos concretos de buena y mala práctica.

## Función

- Aterrizar conceptos.
- Ilustrar decisiones de diseño.
- Hacer visible la diferencia entre profundidad y superficialidad.

## Contenido futuro

- Ejemplos de etapas profesionales.
- Ejemplos de buenas preguntas.
- Ejemplos de malas preguntas.
- Ejemplos de competencias con evidencia.
- Ejemplos de validación con el usuario.

## Relación con el playbook

Los ejemplos ayudarán a comprobar si el playbook funciona en situaciones reales.
